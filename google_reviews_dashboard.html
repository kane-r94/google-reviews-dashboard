Set up manual refresh

A GitHub token is needed to trigger the data update remotely. This is stored only in your browser — it never leaves your device or gets saved to the site.

  1. Go to github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
  2. Click Generate new token
  3. Set repository access to google-reviews-dashboard only
  4. Under Permissions → Repository → Actions: Read and write
  5. Generate and paste the token below
